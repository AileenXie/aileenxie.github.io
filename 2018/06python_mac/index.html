Mac下安装Python3

Written by Aileen.Xie with ♥ on November 19, 2018 in 学习 · 笔记    本文总阅读量次

简单记录mac下Python3安装步骤。

安装Python3

  1. 应先安装C编译器。最快的方式是运行

    xcode-select --install

    来安装Xcode命令行工具。

  2. 安装一个包管理工具Homebrew

    $ /usr/bin/ruby -e " $ (curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

  3. 插入PATH（在~/.profile文件末尾加）

    export PATH=/usr/local/bin:/usr/local/sbin:$PATH

  4. (source .profile 一下使之生效)

  5. 安装Python3

    brew install python

  6. 查看

    python3 --version

  7. 确定pip是否安装

    pip3 --version

    若未安装

    python3 -m ensurepip --default-pip

    pip的使用详见相关参考文档
